GOAL MANAGER (DAILY / WEEKLY / MONTHLY PLANNER)

Goal Manager is a modern GUI-based task management application built using Python and CustomTkinter. The application allows users to organize tasks efficiently with daily, weekly, and monthly planning views. All tasks are automatically saved locally using a JSON file so data is preserved between sessions.

FEATURES

Daily task planner

Weekly goal tracking

Monthly task management

Modern dark-themed UI

Scrollable task list

Mark tasks as completed

Delete tasks easily

Automatic data saving

Keyboard support (Press Enter to add tasks)

Navigation between days, weeks, and months

USER INTERFACE

The application includes a clean and modern interface with rounded task cards, color indicators for completed tasks, sidebar navigation for switching views, and smooth scrolling support.

TECHNOLOGIES USED

Python 3

CustomTkinter

JSON (Local Storage)

Datetime Module

INSTALLATION

Step 1: Clone the repository

git clone https://github.com/ali3618razahacker-dev/gui-task-manager/

Step 2: Install required library

pip install customtkinter

HOW TO RUN

Run the application using:

python main.py

(Replace main.py with your actual Python file name)

DATA STORAGE

All tasks are saved in a local file named:

tasks.json

This file is automatically created when the first task is added.

HOW IT WORKS

Daily Mode
Allows task management based on individual dates.

Weekly Mode
Organizes tasks based on weekly groups.

Monthly Mode
Allows long-term planning by grouping tasks per month.

BUTTON FUNCTIONS

Done Button
Marks a task as completed and changes its color.

Delete Button
Removes the selected task permanently.

Navigation Buttons
Allow switching between previous and next days, weeks, and months.

KEYBOARD SHORTCUT

Press ENTER to quickly add a new task.

SYSTEM REQUIREMENTS

Python 3.8 or higher

CustomTkinter installed

Works on Windows, Linux, and MacOS
